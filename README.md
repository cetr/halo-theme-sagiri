## 安装

#### 压缩包上传方式（推荐）

1. 下载最新发行版zip压缩包到本地 [点击前往](https://github.com/cetr/halo-theme-sagiri/releases)
2. 进入博客后台-外观-主题-安装
3. 选择本地上传，将压缩包上传上去
4. 等待提示安装完成

#### 在线拉取方式

1. 进入博客后台-外观-主题-安装
2. 选择远程下载
3. 远程地址填写 `https://github.com/cetr/halo-theme-sagiri.git`
5. 点击下载，等待下载完成

**主题个性化的配置请在 Sagiri 的主题设置中设置！**
